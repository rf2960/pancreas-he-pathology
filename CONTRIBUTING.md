# Contributing

This repository is primarily a research release for the Reya Lab H&E analysis workflow.

Before contributing:

1. Do not commit raw whole-slide images, large tile datasets, model checkpoints, credentials, or protected project data.
2. Keep code changes reproducible from explicit command-line arguments.
3. Add or update figures/results only when the generating script or source CSV is also included.
4. Use clear commit messages that describe the scientific or engineering change.

For issues, include the command used, the data layout, Python version, GPU/CPU environment, and the relevant traceback or output.
