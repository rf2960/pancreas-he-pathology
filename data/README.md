# Data

Raw data is not stored in this repository. See [docs/data.md](../docs/data.md) for access notes and the expected local folder structure.

Place local exported tiles here only for private experiments, or pass their location explicitly:

```bash
python src/he_ml_pipeline.py --data_dir /path/to/spatial_tiles_dataset
```

Do not commit raw slides, full tile exports, credentials, or protected project data.
